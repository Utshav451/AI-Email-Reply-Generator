# 📧 AI Email Reply Generator

An AI-powered web application that generates professional email replies using Google's Gemini API. The application allows users to paste an email, select a desired tone, and instantly receive a context-aware reply generated by AI.

---

## 🚀 Features

* Generate email replies using Generative AI
* Multiple reply tones:

  * Professional
  * Friendly
  * Casual
  * Sarcastic
* Clean and responsive Material UI interface
* RESTful Spring Boot backend
* Real-time API communication using Axios
* Clipboard copy functionality
* Cross-origin support for frontend-backend communication

## ⚙️ How It Works

### Step 1

User enters the original email content.

### Step 2

User selects an optional tone.

Examples:

* Professional
* Friendly
* Casual
* Sarcastic

### Step 3

Frontend sends a POST request to:

http://localhost:8080/api/email/generate

Request Body:

```json
{
  "emailContent": "Original email content",
  "tone": "Professional"
}
```

### Step 4

Spring Boot receives the request and creates a prompt for Gemini.

### Step 5

Gemini API generates a contextual email response.

### Step 6

Generated reply is returned to the React application and displayed to the user.

## 🔑 Gemini API Configuration

Add the following properties to your `application.properties` file:

```properties
gemini.api.url=YOUR_GEMINI_API_URL
gemini.api.key=YOUR_GEMINI_API_KEY
```

Example:

```properties
gemini.api.url=https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent
gemini.api.key=YOUR_API_KEY
```

## ▶️ Running the Backend

### Clone Repository

```bash
git clone https://github.com/Utshav451/AI-Email-Reply-Generator.git
```

### Navigate to Backend

```bash
cd backend
```

### Install Dependencies

```bash
mvn clean install
```

### Run Application

```bash
mvn spring-boot:run
```

Backend will start at:

```text
http://localhost:8080
```

---

## 📌 API Endpoint

### Generate Email Reply

```http
POST /api/email/generate
```

Request:

```json
{
  "emailContent": "Can you send me the project status update?",
  "tone": "Professional"
}
```

Response:

```text
Thank you for your email. I appreciate your inquiry regarding the project status. The project is progressing according to schedule...
```
